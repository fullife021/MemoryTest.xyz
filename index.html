About Memory Test

Memory Test is a challenging cognitive game designed to help you improve your memory capacity through progressive difficulty levels. Whether you're a student looking to enhance your study abilities or a professional aiming to maintain mental sharpness, this game offers an engaging way to train your brain.

Progressive Challenge

Start with single digits and progress up to 20-digit numbers as you advance through the levels.

Time Pressure

5 seconds to memorize each number, testing both your short-term memory and focus.

Instant Feedback

Get immediate feedback on your performance with our color-coded progress tracking system.

How to Play

Game Rules

  • The game consists of 20 levels
  • Each level presents a number for 5 seconds
  • Numbers increase in length with each level (1 digit to 20 digits)
  • Memorize the number within the time limit
  • Input your answer when the number disappears
  • Progress through all levels to test your memory capacity

Scoring System
